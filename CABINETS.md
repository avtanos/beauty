# Кабинеты мастера и клиента

## Кабинет мастера (`/professional`)

### Функционал:

1. **Дашборд**
   - Статистика услуг (количество активных)
   - Статистика бронирований (всего, завершено, ожидает)
   - Общие доходы
   - Рейтинг и количество отзывов

2. **Управление услугами**
   - Просмотр всех своих услуг
   - Создание новой услуги
   - Редактирование услуги
   - Активация/деактивация услуги

3. **Бронирования**
   - Просмотр всех бронирований
   - Фильтрация по статусам
   - Управление статусами:
     - Подтвердить (pending → confirmed)
     - Начать (confirmed → in_progress)
     - Завершить (in_progress → completed)

4. **Отзывы**
   - Просмотр всех отзывов о мастере
   - Рейтинг и комментарии клиентов

### API Endpoints:

- `GET /api/professional/stats` - Статистика мастера
- `GET /api/professional/services` - Список услуг мастера
- `POST /api/professional/services` - Создать услугу
- `PUT /api/professional/services/{id}` - Обновить услугу
- `DELETE /api/professional/services/{id}` - Деактивировать услугу
- `GET /api/professional/bookings` - Список бронирований
- `PUT /api/professional/bookings/{id}/status?status={status}` - Обновить статус
- `GET /api/professional/reviews` - Список отзывов

## Кабинет клиента (`/client`)

### Функционал:

1. **Дашборд**
   - Статистика заказов (всего, завершено, ожидает)
   - Общая сумма потраченных средств
   - Количество оставленных отзывов
   - Активность за последний месяц

2. **Мои заказы**
   - Просмотр всех бронирований
   - Детальная информация о каждом заказе
   - Возможность оставить отзыв для завершенных заказов

3. **Мои отзывы**
   - Просмотр всех оставленных отзывов
   - История оценок и комментариев

4. **Избранные мастера**
   - Список мастеров с высоким рейтингом (≥4.8)
   - Быстрый переход к профилю мастера

### API Endpoints:

- `GET /api/client/stats` - Статистика клиента
- `GET /api/client/bookings` - Список бронирований
- `GET /api/client/bookings/{id}` - Детали бронирования
- `GET /api/client/reviews` - Список отзывов клиента
- `GET /api/client/favorites/professionals` - Избранные мастера

## Навигация

В навбаре для авторизованных пользователей отображаются ссылки:
- **Клиенты**: "Мой кабинет" → `/client`
- **Мастера**: "Кабинет мастера" → `/professional`
- **Администраторы**: "Админ-панель" → `/admin`

## Доступ

Кабинеты доступны только авторизованным пользователям с соответствующей ролью:
- Кабинет мастера: только для `role = 'professional'`
- Кабинет клиента: только для `role = 'client'`
