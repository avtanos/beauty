# Устранение проблем с отображением интерфейса

## Серверы запущены

✅ **Backend (FastAPI)**: http://localhost:8080  
✅ **Frontend (React/Vite)**: http://localhost:5174

## Как открыть интерфейс

1. Откройте браузер (Chrome, Firefox, Edge)
2. Перейдите по адресу: **http://localhost:5174**
3. Если не работает, попробуйте: **http://127.0.0.1:5174**

## Если интерфейс не отображается

### Шаг 1: Проверьте консоль браузера
1. Нажмите **F12** или **Ctrl+Shift+I**
2. Откройте вкладку **Console**
3. Проверьте наличие ошибок (красные сообщения)
4. Скопируйте текст ошибок

### Шаг 2: Проверьте вкладку Network
1. В консоли браузера откройте вкладку **Network**
2. Обновите страницу (F5)
3. Проверьте, есть ли запросы с ошибками (красные)
4. Особенно проверьте запросы к `/api/`

### Шаг 3: Проверьте, что серверы работают
1. Откройте в браузере: http://localhost:8080/docs
   - Должна открыться документация API
2. Откройте: http://localhost:8080/api/health
   - Должен вернуться JSON: `{"status":"healthy"}`

### Шаг 4: Очистите кэш браузера
1. Нажмите **Ctrl+Shift+Delete**
2. Выберите "Кэшированные изображения и файлы"
3. Нажмите "Очистить данные"
4. Обновите страницу (Ctrl+F5)

### Шаг 5: Проверьте файлы
Убедитесь, что все файлы на месте:
- `frontend/index.html`
- `frontend/src/main.jsx`
- `frontend/src/App.jsx`
- `frontend/src/index.css`

## Возможные ошибки

### Ошибка: "Cannot find module"
**Решение**: Запустите `npm install` в папке `frontend`

### Ошибка: "CORS error"
**Решение**: Проверьте, что backend запущен на порту 8080

### Ошибка: "Network Error"
**Решение**: 
1. Проверьте, что backend запущен
2. Проверьте файл `frontend/vite.config.js` - там должен быть proxy на `/api`

### Белый экран
**Решение**:
1. Откройте консоль браузера (F12)
2. Проверьте ошибки
3. Попробуйте открыть http://localhost:5174/#/

## Контакты для отладки

Если проблема не решена, предоставьте:
1. Скриншот консоли браузера (F12 → Console)
2. Скриншот вкладки Network
3. Текст ошибок из консоли
