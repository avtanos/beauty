# Админ-панель - Руководство

## Доступ

**URL:** http://localhost:5173/admin

**Учетные данные администратора:**
- Email: `admin@example.com`
- Пароль: `admin123`

## Функционал

### 1. Дашборд
Общая статистика системы:
- Количество пользователей (клиенты, мастера)
- Количество услуг
- Статистика бронирований
- Количество отзывов
- Общие доходы
- Средний рейтинг мастеров
- Бронирования по статусам

### 2. Управление пользователями
- Просмотр всех пользователей
- Поиск по имени, email, телефону
- Активирование/деактивирование пользователей
- Просмотр информации о пользователях (роль, рейтинг, статус)

### 3. Управление услугами
- Просмотр всех услуг
- Фильтрация по категориям
- Активирование/деактивирование услуг
- Просмотр информации об услугах (цена, длительность, мастер)

### 4. Управление бронированиями
- Просмотр всех бронирований
- Фильтрация по статусам
- Просмотр детальной информации о бронированиях

### 5. Управление отзывами
- Просмотр всех отзывов
- Удаление отзывов
- Автоматический пересчет рейтинга мастера при удалении отзыва

## API Endpoints

Все endpoints требуют прав администратора:

### Статистика
- `GET /api/admin/stats` - Общая статистика

### Пользователи
- `GET /api/admin/users` - Список пользователей (с фильтрацией)
- `GET /api/admin/users/{id}` - Детали пользователя
- `PUT /api/admin/users/{id}` - Обновить пользователя
- `DELETE /api/admin/users/{id}` - Деактивировать пользователя
- `POST /api/admin/users/{id}/activate` - Активировать пользователя

### Услуги
- `GET /api/admin/services` - Список услуг (с фильтрацией)
- `DELETE /api/admin/services/{id}` - Деактивировать услугу
- `POST /api/admin/services/{id}/activate` - Активировать услугу

### Бронирования
- `GET /api/admin/bookings` - Список бронирований (с фильтрацией)
- `PUT /api/admin/bookings/{id}` - Обновить бронирование

### Отзывы
- `GET /api/admin/reviews` - Список отзывов (с фильтрацией)
- `DELETE /api/admin/reviews/{id}` - Удалить отзыв

## Безопасность

- Все endpoints защищены проверкой прав администратора
- Администратор не может деактивировать сам себя
- Только пользователи с ролью `admin` имеют доступ к админ-панели

## Навигация

В навбаре для администраторов отображается ссылка "Админ-панель" с иконкой щита.
