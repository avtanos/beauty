# Мок-данные для разработки

Этот модуль содержит мок-данные для тестирования интерфейса без подключения к backend.

## Использование

Мок-данные автоматически включаются в режиме разработки (`development`).

### Включение/выключение мок-данных

В консоли браузера (F12):

```javascript
// Включить мок-данные
localStorage.setItem('useMockData', 'true')
location.reload()

// Выключить мок-данные (использовать реальный API)
localStorage.setItem('useMockData', 'false')
location.reload()
```

## API функции

Все мок-функции находятся в `mocks/api.js`:

- `getServices(params)` - получить список услуг
- `getService(id)` - получить услугу по ID
- `getProfessionals(params)` - получить список мастеров
- `getProfessional(id)` - получить мастера по ID
- `getBookings()` - получить бронирования
- `getReviews(params)` - получить отзывы

## Примечания

- Мок-данные имитируют задержку сети (300-500ms)
- Все функции возвращают данные в том же формате, что и реальный API
- Мок-данные работают только для GET запросов
- POST/PUT/DELETE запросы будут отправляться на реальный API
